\# AI PDF Chatbot Analyzer



AI-powered PDF Chatbot built using Python, Streamlit, ChromaDB, and Groq API.



This project allows users to upload PDF files, extract content, store embeddings, and ask contextual questions using Retrieval-Augmented Generation (RAG).



\---



\## Features



\- Upload and analyze PDF documents

\- AI-based contextual question answering

\- Vector storage using ChromaDB

\- Fast responses using Groq API

\- Interactive Streamlit interface



\---



\## Tech Stack



\- Python

\- Streamlit

\- ChromaDB

\- LangChain

\- Groq API

\- Sentence Transformers
